# Headphones Project

## Overview
This project is a responsive web page designed for showcasing headphones. The design is created in Figma and adapts seamlessly to different screen sizes.

## Figma Design
- Access the design on Figma: [Figma Project Link](#)  
- Fonts Used:  
  - [Source Sans Pro](https://fonts.google.com/specimen/Source+Sans+Pro)
  - [Spin Cycle OT](https://github.com/Skill-Labs/Spin-Cycle)

## Key Features
- Responsive design that switches to mobile view for screens 480px or less.
- Interactive elements:
  - Links change color on hover/active: `#FF6565`.
  - Buttons change opacity on hover/active: `0.9`.
  
## Layout Specifications
- Maximum content width: `1000px`, centered on the page.
  
## Getting Started
1. Clone the repository .
2. Open the project files in your preferred editor.
3. Make any necessary modifications based on the Figma design.
4. Test responsiveness by adjusting screen width.
  
## Deployment
- Ensure the website is functional in multiple browsers.
- Push your changes back to GitHub when complete.
